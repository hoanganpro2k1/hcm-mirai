# AI Rules

## Ngôn ngữ

- Luôn trả lời bằng tiếng Việt
- Không sử dụng tiếng Anh trừ khi là từ khóa kỹ thuật bắt buộc
- Giải thích code bằng tiếng Việt

## Phong cách trả lời

- Ngắn gọn, rõ ràng, đi thẳng vào vấn đề
- Ưu tiên ví dụ thực tế
- Tránh giải thích dài dòng

## Code

- Comment trong code phải bằng tiếng Việt
- Ưu tiên best practice
- Viết code dễ đọc, dễ maintain

## Ưu tiên công nghệ

- Backend: NestJS
- Frontend: Next.js
- Database: PostgreSQL hoặc MySQL
